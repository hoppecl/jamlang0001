# Explaining

### Datatypes
* Comment
	* the most important type
* Number
	* floating point numbers
* String
	* unicode strings
* Bool
	* `true`or `false`
*  Unit
	* only the value `()`
    

### Syntax
The syntax is a C like, but does not use semicolons to separate statements. To avoid ambiguity only some expression are allowed as statements (similar to Lua).
